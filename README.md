# express-api


Environment

1.Database Mongodb version v2.4.9
2.Nodejs v6.10.2
3.Npm v3.10.10

Intall npm dependency sebelum memulai

    npm install

Ubah database config in app.js, baris 14

    mongoose.connect('mongodb://localhost/test');

Jalankan Program

    npm start

Lakukan test pada aplikasi

    npm test
